# Email Spam Detection

## Project Overview
This project is developed for the COM763 Advanced Machine Learning module.

The application classifies emails as **Spam** or **Ham (Not Spam)** using Machine Learning.

## Machine Learning Algorithm
- Multinomial Naive Bayes

## Feature Extraction
- TF-IDF Vectorizer

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- NLTK
- Pandas
- NumPy
